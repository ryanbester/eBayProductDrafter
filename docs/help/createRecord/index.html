Help > Creating a new record

Creating a new record

  1. Open eBay Product Drafter
  2. Click the New button on the toolbar, or select File and then New
  3. Enter a title for the product
  4. Select a condition for the product
  5. Describe the condition for the product
  6. Describe the product
  1. Click the Check SKU button
  2. Enter an SKU into the box
  3. Click Search
  4. If the SKU already exists, the exisiting SKU will be highlighted. If the SKU does not exist, click the OK button
  1. Enter a price for the product
  2. Click the Save button on the toolbar or select File and then Save
  3. If the record saves succcessfully, a dialog will appear asking you to select an option. Select Create a New Record to repeat this process, Edit an Existing Record to edit a record which already exists, or Exit

Errors

Enter an SKU

This error will occur when an SKU has not been entered into the SKU field.

To fix this error:

  • Check that the SKU field contains a number
  • Make sure that there are no spaces or invisible characters in the SKU field. To check this, right click inside the SKU field and select Show Unicode control characters.

File already exists

This error occurs when the SKU entered into the SKU field has already been used.

To fix this error:

  • Use the Check SKU feature to choose an SKU which has not already been used.
